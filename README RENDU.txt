Fichier principale devant être éxécuté sur le robot : distanceMatrixTank.py

Execution du script d'affichage visuel des donnée des scan (Les données fournit sont celles de l'un des scan réalisé durant la séance de présentation) :

- Mettre les données (distanceData1.txt, distanceData2.txt) dans le même dossier que distanceReader.py
- Ouvrir un terminal au dossier, puis éxécuter 'python3 distanceReader.py'
- Laisser Turtle s'exécuter, elle restera active 30 secondes.
