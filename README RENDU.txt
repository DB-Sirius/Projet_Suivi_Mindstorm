Les données du scan filmé sont fournies : ce sont les données du tout dernier scan terminé (les données sont écrasées entre chaque itération), correspondant aux fichiers distanceData1 et distancedata2.
Un fichier tableauDiff ser aéventuellement fourni, mais il n'est pas utilisé, il est là pour du débug et correspond aux séries de différences significatives trouvées.


LECTURE DES DONNÉES DE SCAN:

Mettre les données (distanceData1.txt, distanceData2.txt) dans le même dossier que distanceReader.py
Ouvrir un terminal au dossier, puis éxécuter 'python3 distanceReader.py'
Laisser Turtle s'exécuter, elle restera active 30 secondes.
